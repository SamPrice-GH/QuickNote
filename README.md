# QuickNote

A wearble note-taking device for hands-free short note capture.

See [system_architecture.md](https://github.com/SamPrice-GH/QuickNote/blob/main/device/docs/system_architecture.md).

This is a personal project that is at the developmental stage of a new-born. Still very much prototyping. Developed for my tradesmen friends/family who can't seem to figure out existing smart watches and complain that Siri never works.
